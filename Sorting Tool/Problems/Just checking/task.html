Just checking

Write a program that reads an unsorted array of integers and two numbers p and m. The program is to check if p and m occur next to each other in the array (in any order).

The first line contains the size of an array, N.
The next N lines contain elements of the array.
The last line contains two integer numbers p and m, separated by the space character.

Output "YES" or "NO".


Sample Input:
3
1
3
2
2 3

Sample Output:
YES


Sample Input:
3
1
2
3
3 4

Sample Output:
NO



Memory limit: 256 MB
Time limit: 8 seconds
